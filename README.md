## 在本地启动该项目

### `git clone`

```bash
git clone git@github.com:busyzz-1994/busyzz-1994.github.io.git
```

### `yarn install`

```bash
yarn install
```

### `yarn start`

```bash
yarn start
```

[http://localhost:8080](http://localhost:8080/) 访问
